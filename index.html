Elemento Âncora, o <a>

Essa tag é uma âncora para outras páginas e tem uma séria de atributos. Uma tag a pode ter um link URL, um email e até um telefone. Pode ser do tipo Download, do tipo self, em que o link abre na aba atual, e blank, em que o link abre numa nova guia.

Sobre a âncora:

para navegar nos diretórios do documento fonte do site escreva "nomeDocumento.html" ou "diretório/nomeDocumento.html" para entrar num diretório.
para sair escreva "../nomeDocumento.html" e assim estará num diretório acima.
Para o diretório raiz escreva "./nomeDocumento.html".

No caminho absoluto inclui-se o protocolo e o nome de domínio e sempre apontará para o mesmo local, já que é absoluto. Como por exemplo esta URL http://www.rocketseat.com.br/projeto/index.html é absoluto(mesmo não levando a lugar nenhum, é apenas um exemplo).
No caminho relativo, é relativo à página, pasta aberta, ou seja, se não tiver irá consequentemente dar erro, apontará para lugares diferentes. Como por exemplo apenas google.com.
